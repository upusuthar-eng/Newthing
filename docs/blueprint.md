# **App Name**: Text Display

## Core Features:

- Text Rendering: Display the text 'New App' on the screen.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calm and inviting feel.
- Background color: Very light blue (#F0F8FF), almost white, complements the primary color.
- Accent color: Light violet (#BDB2FF) for interactive elements.
- Font: 'Inter' sans-serif font for clear, modern readability.
- Center the text 'New App' both horizontally and vertically on the screen for optimal visibility.
- Fade in the text smoothly upon loading the app for a gentle introduction.
